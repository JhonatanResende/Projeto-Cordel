## 📌 Sobre o projeto
Projeto desenvolvido com HTML e CSS com o objetivo de praticar estilização moderna e efeitos visuais, como o efeito Parallax.

Inspirado no Cordel Moderno, o projeto busca unir poesia e tecnologia em uma experiência visual agradável.

## 🚀 Tecnologias utilizadas
- HTML (estrutura da página)
- CSS (estilização, responsividade e efeitos visuais)

## 🎯 Funcionalidades
- Efeito Parallax
- Layout responsivo
- Tipografia estilizada
- Uso de imagens de fundo
